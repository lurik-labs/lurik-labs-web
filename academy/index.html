Syllabus

Course: Game Development with Unity (C# Programming)
Instructor(s): Luis Angel Medina

Course Overview: In this course we will be building games from start to finish. Students will be learning about Unity’s interface, and improving their programming skills with game programming. By the end of this course students will start on their own projects.

Course Goals:
Navigate Unity’s interfaces
Learn gameplay programming with C#
Create 2 executable games
Start on personal projects

Course Duration: 8 sessions, 90 minutes each
Course Dates: Every Tues/Thurs, 3/5/2019 - 3/28/2019

Software Requirements
Unity (https://unity3d.com/get-unity/download)
Text Editor/IDE (Recommended: Visual Studio 2017, comes with Unity)

Session 1: Setting up Unity / GameObjects / Scripts
Session 2: Input / Prefab and Spawning
Session 3: Physics / Collision / UI
Session 4: Particle Effects / Animation / Asset Store
Session 5: Main Menu / Game Over / Building Executable
Session 6: Platformer Movement with Jump
Session 7: Shooting / Spawn and Kill Enemies
Session 8: Start Final Project
